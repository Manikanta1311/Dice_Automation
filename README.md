# Dice_Automation
Automate job applications on Dice.com using Python and Selenium

Features
- Auto login to Dice.com
- Search jobs by keyword and location
- Automatically applies to “Easy Apply” jobs
- Handles multiple pages (pagination)
- Avoids duplicate job applications
- Uses a JSON config file for flexible setup

Tech Stack
- Python 3.8+
- Selenium WebDriver
- WebDriver Manager
- JSON Configuration
